# Личный кабинет финансов

Простой сайт на HTML/CSS/JS для учета личных финансов с сохранением данных в `localStorage`.

## Возможности
- Добавление доходов и трат с категорией, суммой, датой и комментарием.
- Фильтр по месяцу.
- Автоматический расчет: доходы, траты, остаток.
- История операций с удалением записей.
- Адаптивный интерфейс для телефона.
- Данные сохраняются в браузере между сессиями.

## Запуск
Открой `index.html` в браузере или запусти локальный сервер:

```bash
python3 -m http.server 4173
```

И перейди на `http://localhost:4173`.
